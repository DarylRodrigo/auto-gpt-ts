# Autonomous GPT

### Getting started

1. Install the package

```bash
npm install
```

2. Create a `.env` file in the root directory and add the following

- Copy paste the .env.example file 
- Rename it to .env
- Fill in the values

3. Start Docker

```bash
npm run docker-compose
```

4. Run the example

```bash
ts-node src/main.ts
```
